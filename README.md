# 鞋程

## 心得总结
这是一个很小的期末试训项目，用的是传统的servlet/jsp进行后台编码，数据访问层是通过自己封装的一个工具类来进行交互。最大的收获是完整的体验了一次开发流程，虽然其中有些环节是浑水摸鱼，但是对于企业产品级的开发有了初步的认知。在为期两周的时间里，我感受到了团队开发的重要性，体会到了从未有过的开发氛围。
